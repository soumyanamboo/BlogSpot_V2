Components of the Project

FrontEnd

  1. User Login (Token based authentication) / Register New User
  2. Feeds
  3. User Profile
  4. User Management
    • Update or Delete User
  5. Post Management
    • Add, Update, Delete Posts
  6. Search for Users
  7. Follow / Unfollow Users

API

  1. CRUD for User
  2. CRUD for Posts
  3. CRUD for Comments
  4. Search User
  5. Follow / Unfollow User
  6. Users feeds

Backend Jobs

  1. Export Posts of a User - User triggered Async Job
  2. Daily Remainder Job - Scheduled Job
  3. Monthly User Report - Scheduled Job
